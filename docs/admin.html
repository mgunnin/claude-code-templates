Plugin Information

Note: Plugins are collections of other components (agents, commands, MCPs, hooks) defined in .claude-plugin/marketplace.json.

To create a plugin, you need to manually edit the marketplace.json file. Plugins bundle multiple components together.

See .claude-plugin/marketplace.json for examples of plugin definitions.
